# Competition Notes

本文件用于记录比赛规则、关键限制和算法设计时必须满足的条件。

## 任务类型

感觉肌肉电刺激辅助运动想象脑电分类任务。当前项目暂不绑定具体算法，先建立统一的数据、预处理、训练、推理和评估流程。

## 关键约束

| 项目 | 约束 |
|---|---|
| 单试次窗口 | 不超过 4 s |
| 单次推理时间 | 小于 1 s |
| EEG 通道数 | 不超过 8 个 |
| 模型大小 | 不超过 150 MB |
| 校准数据 | 每类 trial 数量有限 |
| 外部数据 | 不使用非开源数据训练或校准 |

## 初步算法设计原则

1. 优先建立可解释、轻量、稳定的传统机器学习基线。
2. 优先控制通道数、模型大小和推理时间，而不是盲目堆叠复杂模型。
3. 数据量较小时，避免直接使用参数量较大的深度网络。
4. 实验结果必须记录随机种子、通道选择、时间窗、滤波频段、特征类型和分类器参数。
5. 后续所有算法改动应通过统一评估脚本比较，避免只凭单次提交分数判断优劣。
